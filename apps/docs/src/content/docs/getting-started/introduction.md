# Minions Qa-suites

Test suite definitions, test cases, and assertions for validating clawspace toolboxes

## Overview

This project is part of the [Minions ecosystem](https://github.com/mxn2020/minions) and provides:

- **TypeScript SDK** (`@minions-qa-suites/sdk`) — Core library for Node.js
- **Python SDK** (`minions-qa-suites`) — Python bindings
- **CLI Tool** (`@minions-qa-suites/cli`) — Command-line interface
- **Web App** — Interactive playground at [qa-suites.minions.wtf](https://qa-suites.minions.wtf)
